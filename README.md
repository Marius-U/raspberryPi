# raspberryPi
Collection of various raspberry pi based mini-projects.

## For the rapsberry Pi zero w v1.1:
### Raspberry Pi pin selection for L298N H bridge controll:
**IN1 - GPIO 22**  
**IN2 - GPIO 23**  
**IN3 - GPIO 24**  
**IN4 - GPIO 25**  
**ENA - GPIO 12 (PWM)**  
**ENB - GPIO 13 (PWM)**
